## General info
This project is CRUD operations of the superhero model.
	
## Technologies
Project is created with:
* React
* React-Router-Dom
* Axios
* Node
* Express
* Sequelize
	
## Setup
To run this project
1) Install node dependencies using npm:
```bash
$ cd server/
$ npm install
$ npm run migrate
$ node app.js
```
2) Install react dependencies using npm:
```bash
$ cd ../client/
$ npm install
$ npm start
```
